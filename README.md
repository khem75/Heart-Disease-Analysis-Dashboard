📌 Overview

This project presents an interactive Heart Disease Analysis Dashboard built using Microsoft Power BI. The dashboard helps analyze patient survival patterns and key medical indicators associated with heart failure.

🚀 Features
📊 KPI Cards
Survival Rate
Average Age of Survivors
Total Survivors
Total Deaths
📈 Visualizations
Survival Count by Age Group
Average Serum Creatinine by Age
Average Ejection Fraction Analysis
Survival Rate Trend Line
Risk Factor Ribbon Chart
Gender-based Filtering
🎛 Interactive Elements
Slicers
Dynamic Filtering
Responsive Dashboard Layout
🛠 Tools Used
Microsoft Power BI
Canva (Dashboard Background)
Excel Dataset
DAX
Power Query
📂 Dataset

The dataset used in this project comes from the UCI Machine Learning Repository.

Dataset: Heart Failure Clinical Records

🎯 Key Insights
Survival rates decrease with increasing age.
Serum creatinine levels correlate with patient outcomes.
Ejection fraction significantly impacts survival probability.
Diabetes, smoking, and high blood pressure contribute to risk.
